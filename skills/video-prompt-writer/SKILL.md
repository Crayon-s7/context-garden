# Video Prompt Writer Skill

## 目标

编写高质量的 AI 视频生成提示词，精准控制画面、运动、光线和氛围。

## 适用场景

- AI 视频生成（Sora、Runway、可灵、Pika 等）
- 视频分镜脚本
- 广告/宣传片 AI 生成
- 社交媒体视频内容

## 输出原则

1. **结构化描述**：按景别→运镜→主体→环境→光线→氛围的顺序。
2. **一次一个核心动作**：避免描述过多动作导致 AI 混乱。
3. **用摄影语言**：用专业的摄影/电影术语，效果更精确。
4. **光线和氛围不可省略**：这是视频质感的关键。

## 提示词结构

```text
[景别] [运镜] [主体描述] [主体动作] [环境/场景] [光线] [氛围/情绪] [技术参数]
```

## 字段参考

| 字段 | 常用值 |
|---|---|
| 景别 | 特写 / 近景 / 中景 / 全景 / 远景 / 航拍 |
| 运镜 | 固定 / 推进 / 拉远 / 平移 / 环绕 / 跟随 / 升降 |
| 光线 | 自然光 / 侧光 / 逆光 / 暖色调 / 冷色调 / 黄金时刻 |
| 氛围 | 温馨 / 紧张 / 孤独 / 活力 / 神秘 / 怀旧 |
| 技术参数 | 4K / 电影质感 / 浅景深 / 24fps / 胶片质感 |

## 输出格式

对每个镜头输出：

```markdown
### 镜头 N

**提示词**:
\`\`\`
（完整英文或中文提示词）
\`\`\`

**说明**: 这一刻画面在表达什么、为什么这样设计。

**建议时长**: X 秒
```

## 不同工具的差异

| 工具 | 语言偏好 | 运镜支持 | 特点 |
|---|---|---|---|
| Sora | 英文 | 丰富 | 物理理解强 |
| Runway | 英文 | 中等 | 风格化强 |
| 可灵 | 中文/英文 | 中等 | 中文场景理解好 |

- "为什么生成的画面和描述不符？"
- "怎么写长视频的连续镜头？"
