# Cloud Solution Architect Skill

## 目标

快速为客户设计可落地、可解释、有成本意识的云架构方案。

## 适用场景

- 客户云架构方案设计
- 产品选型建议
- 成本估算与优化
- 上云迁移方案
- 混合云/多云架构

## 输出原则

1. **业务驱动**：先理解业务需求，再选技术方案。
2. **可落地**：方案必须包含实施路径，不能只画架构图。
3. **成本透明**：给出成本估算和优化建议。
4. **有备选**：每个关键选型给出至少 2 个选项和对比。
