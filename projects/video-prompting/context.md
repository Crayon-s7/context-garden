# 视频提示词 - 项目上下文

## 一句话定位

系统性地研究和编写视频生成提示词（AI 视频工具如 Sora、Runway、可灵等）。

## 核心目标

- 建立一套可复用的视频提示词编写方法论。
- 积累不同场景的提示词模板。
- 总结不同 AI 视频工具的 prompt 差异。

## 当前阶段

- 正在积累镜头级别的提示词模板。
- 已有一些实验性成果，需要结构化整理。

## 关键发现

- 视频提示词和图片提示词不同，需要描述运动和时间变化。
- 摄影语言（景别、运镜、光线）对效果影响很大。
- 不同工具对同一 prompt 的理解和表现差异显著。

## 下一步计划

- [ ] 整理 10 个典型场景的提示词模板。
- [ ] 建立景别 + 运镜 + 主体 + 环境的提示词结构。
- [ ] 测试同一 prompt 在不同工具上的表现差异。
