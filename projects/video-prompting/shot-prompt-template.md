# 视频分镜提示词模板

> 一个标准化的视频提示词结构，适用于大多数 AI 视频生成工具。

## 提示词结构

```text
[景别] [运镜] [主体描述] [主体动作] [环境/场景] [光线] [氛围/情绪] [技术参数]
```

## 字段说明

| 字段 | 说明 | 示例 |
|---|---|---|
| 景别 | 画面范围 | 特写 / 近景 / 中景 / 全景 / 远景 / 航拍 |
| 运镜 | 镜头运动 | 固定 / 推进 / 拉远 / 平移 / 环绕 / 跟随 |
| 主体描述 | 画面核心对象 | 一位穿白衬衫的年轻女性 |
| 主体动作 | 主体在做什么 | 低头看手机，微微皱眉 |
| 环境/场景 | 背景和场景 | 现代办公室，落地窗外是城市夜景 |
| 光线 | 光照条件 | 暖色调侧光，窗外霓虹光映在脸上 |
| 氛围/情绪 | 整体感觉 | 紧张、专注、略带焦虑 |
| 技术参数 | 画面质量 | 4K, 电影质感, 浅景深 |

## 完整示例

```text
中景 缓慢推进 一位穿黑色风衣的男人 站在雨中的十字路口，抬头望向远方 
城市街道，霓虹灯倒映在湿漉漉的路面 冷色调，路灯光穿过雨滴 
孤独、沉思、悬念感 电影质感，4K，浅景深，24fps
```

## 注意事项

- 先写画面，再写运动，最后写技术参数。
- 避免同时描述太多动作，AI 视频工具一次最好只表现一个核心动作。
- 光线和氛围对最终效果影响最大，不要省略。
